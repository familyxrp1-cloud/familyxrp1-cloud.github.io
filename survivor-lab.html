Extracted heuristics

The common denominators, as rules.

Each rule shows the win-rate multiplier when the condition holds, with the sample size behind it. Computed live from the dataset.

Basket math · buy 5, need 1

Your strategy, quantified against this dataset. Each filter level changes the hit rate — and the hit rate changes the odds that a 5-token basket contains at least one survivor.

How to read this: a 3.4× lift means launches matching the condition survived 3.4 times more often than the population average. Lift is correlation, not causation — a deployer with aged wallets and real socials is also the kind of deployer who doesn't rug. Sample sizes below ~10 are noise; treat them as hypotheses, not signals. None of this is financial advice — it's a microscope, not a crystal ball.
Live data pipeline

How the data flows in.

The collector runs 24h unattended and exposes /launches.json. This page fetches it on load and auto-refreshes every 5 minutes.

The data contract

One object per launch. Booleans and numbers only — the engine buckets them itself.

{ name, ticker, mint, bundlePct, // % supply bought in deploy block devAgeDays, // deployer wallet age devPriorRugs, // rugs from linked deployer wallets top10Pct, // holder concentration at grad devSoldEarly, // dev sold >50% pre-grad hasSocials, // X + site live before launch organicBuyers, // unique funded-elsewhere buyers, 30m liqAtGrad, // SOL in pool at graduation outcome, // 'win' | 'rug' | 'fade' peakX, hoursAlive, series, // holders at 0,4,8…48h (13 pts) meta, metaRank, // narrative cluster + nth in it deployerId, // funding-linked wallet cluster smartMoneyBuyers, // profitable wallets in first 30m saleDurationMin, // bonding-curve fill time sameBlockSnipes, // deployer-funded deploy-block buys insiderClusterPct // supply held by funding-linked ring }

Pipeline stages

Three-stage funnel keeps API usage well within free tiers.

Stage 1 PumpPortal WSS → every create + migrate stores mint/name/deployer in SQLite FREE — no cost per token Stage 2 Helius RPC (free key) → graduated only deploy-block txs → bundlePct wallet first-tx date → devAgeDays funding-parent graph → insiderClusterPct, organicBuyers, deployerId DAS getLargestAccounts → top10Pct ~200-300 tokens/day, budget-guarded Stage 3 Dexscreener (free, no key) polled at 12h / 24h / 48h post-grad price + liq → outcome, peakX Helius snapshots every 4h → series
Set window.COLLECTOR_URL before this script loads if the collector runs on a different origin. The server must have CORS enabled (it does by default).
